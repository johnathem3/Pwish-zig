# pwish-zig

Zig port of the pwish-rs project - a personal information display tool.

## Features

- Displays ASCII art avatar
- Shows personal information with colorful output
- Written in Zig for performance and simplicity

## Building and Running

### Prerequisites
- Zig 0.14.1 or later

### Build
```bash
zig build
```

### Run
```bash
zig build run
```

## Project Structure

- `src/main.zig` - Main application code
- `src/pwish.txt` - ASCII art avatar file
- `build.zig` - Build configuration

## Original Project

This is a Zig port of the original Rust version (pwish-rs).
